How to create a virtualenv for web development for Eve and MongoDB

// under python virtualenv web mongodb eve

Installing Web development frameworks

Create virtualenv for Sands (python 2.7.5)

mkvirtualenv -p /usr/bin/python2.7 myweb

Note that you are using the virtualenv: (myweb) in the beggining of the shell line.

pip install ipython[all]

pip install -r requirements.txt
